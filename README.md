# 🚀 AI Lead Qualification & CRM Automation (n8n)

## 📌 Overview

This project is an end-to-end AI-powered automation system built using n8n that captures incoming customer messages, analyzes intent using AI, qualifies leads, and stores them in a CRM automatically.

The system is designed to reduce manual effort, improve response time, and help businesses focus on high-quality leads.

---

## ⚙️ What This Automation Does

* Captures customer messages from Telegram
* Extracts user details (name, message, chat ID)
* Uses AI (OpenAI / Gemini) to understand intent
* Applies lead scoring (Hot / Warm / Cold)
* Stores leads in a PostgreSQL CRM (Supabase)
* Logs data into Google Sheets for reporting
* Sends automated replies to customers
* Notifies sales team for high-priority leads

---

## 🧠 Architecture

Telegram → n8n → AI Processing → Lead Scoring → CRM (PostgreSQL) → Google Sheets → Notification

---

## 🛠 Tech Stack

* n8n (Workflow Automation)
* Telegram Bot API
* OpenAI / Google Gemini (AI Processing)
* Supabase (PostgreSQL Database)
* Google Sheets (Reporting Dashboard)
